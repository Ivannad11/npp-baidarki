# НИЖПРОМПЛАСТ — состояние проекта (сайт байдарок)

> Чтобы продолжить: открой эту папку, прочитай этот файл — здесь всё, что сделано и что дальше.

## Что это
Сайт компании НИЖПРОМПЛАСТ (НПП) — производство пластиковых байдарок **БД-2** (двухместная) и **БД-3** (трёхместная) + комплектующие.
Сделан как **клон референса keelson.shop** (Webflow-экспорт), перекрашенный и переименованный под НПП.

## Структура файлов
```
index.html       — главная (герой + секции + характеристики + сравнение + отзывы)
catalog.html     — каталог с фильтрами Все/Байдарки/Комплектующие
contacts.html    — контакты + форма
optom.html       — для оптовиков
css/webflow.css  — базовый Webflow-CSS (перекрашен: оранжевый #FF8146 → индиго #140C64)
css/brand.css    — мои оверрайды (герой, карточки, адаптив, выравнивание)
css/header.css   — единая шапка + футер .npp-hd / .npp-ft (на всех страницах)
css/site.css     — дизайн-система чистых страниц (каталог/контакты/опт)
assets/          — фото/иконки/шрифты (фото keelson — ПЛЕЙСХОЛДЕРЫ под замену на НПП)
_keelson-clone-backup.html — исходный клон keelson (бэкап, в .gitignore)
```
